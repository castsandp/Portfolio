Box Model

The box model is the concept that each section or html element has 4 parts. One being the contnet part and from there it can have a border. The border and the content are seperated by the padding section. The border of one element and another element are seprated by each elements margin. So it does content, padding, border margin for each element, like multiple boxes growin outwords. I used the box model to help me style the header and main sections. I have both of them some sort of border and neeed to have space between the border and the ocntent so I added padding. I needed hte two borders to be seperated so I added margins to each!
