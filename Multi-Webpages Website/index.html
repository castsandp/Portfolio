Web Dev Concepts

Definitions of terms.

Website: A website is a set of files that is on the internet, visable if the location or the URL is known.

Web Application: A web application provides some sort of functionality to a computer.

Web Browser: A web browser is an application that allows you to connect with different servers that then allows information to be displayed.

Web Server: A web server is software that can communicate with other devices by exchanging information over the internet.

Request: An HTTP request is a request of certain information from the client side to a server.

HTTP Response: An HTTP response is a response from the server to the request sent from the client with information needed and requested.

Frontend Design

Frontend Design is the part of Web-Developemnt that focuses on the look and set up of a website. It uses HTML, CSS, and JavaScript.

Five "E"s of usability

Effectiveness
Creating an effective product.
Efficiency
Getting things done in a timely, and least expensive way.
Engagment
Content is clear and engaging.
Easy to Learn
Product and content can be understood easily.
Error Tolerance
Product can handle user errors and tries to minimize them.

Page Layout Tags

The purpose of page layout tags are to give a web page structure and organization!

Tag Description
Paragraph Inputs text as a paragraph. It will be given a paragraph font and size.
Header Inputs text with header font and size.
Link Inputs a link.
Ordered List Inputs a numbered list.
Unordered List Inputs an unordered or bulleted list.
Divisions Groups content.
